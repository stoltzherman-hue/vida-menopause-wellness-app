# Environment Variable Template

Create `.env.local` from this list.

```bash
# App
NEXT_PUBLIC_APP_URL=http://localhost:3000
NEXT_PUBLIC_APP_NAME=Vida
NODE_ENV=development

# Database
DATABASE_URL=
DIRECT_DATABASE_URL=

# Supabase optional
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

# Auth - Clerk or NextAuth
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=

# AI provider
AI_PROVIDER=openai
AI_MODEL=
OPENAI_API_KEY=
ANTHROPIC_API_KEY=
AI_MAX_MONTHLY_FREE_MESSAGES=10
AI_MAX_MONTHLY_PREMIUM_MESSAGES=500

# Voice phase 2
AI_VOICE_PROVIDER=
AI_VOICE_MODEL=

# Stripe
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET=
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
STRIPE_PRICE_MONTHLY=
STRIPE_PRICE_ANNUAL=

# Email
RESEND_API_KEY=
SENDGRID_API_KEY=
EMAIL_FROM=Vida <hello@example.com>

# Security
ENCRYPTION_KEY=
AUDIT_LOG_SALT=
RATE_LIMIT_REDIS_URL=

# Notifications
NEXT_PUBLIC_VAPID_PUBLIC_KEY=
VAPID_PRIVATE_KEY=
VAPID_SUBJECT=mailto:hello@example.com

# Analytics
ANALYTICS_PROVIDER=none
POSTHOG_KEY=
POSTHOG_HOST=
```

Never expose secret keys in client components.
