# Launch Readiness Checklist

## Product

- Onboarding complete.
- Tracker complete.
- Dashboard complete.
- AI companion safe for MVP.
- Community guidelines visible.
- Report/export complete.
- Billing complete.

## Safety

- Red-flag tests pass.
- HRT refusal tests pass.
- Self-harm tests pass.
- Community moderation flow tested.
- Disclaimers in place.

## Privacy

- Privacy policy drafted.
- Terms drafted.
- Consent flows working.
- Export working.
- Deletion working.
- AI memory control working.
- No PHI in logs.

## Security

- RLS/ownership tests pass.
- Admin roles tested.
- Stripe webhook signature verified.
- Rate limits active.
- Secrets not exposed.
- Security headers configured.

## Accessibility

- Keyboard navigation checked.
- Screen reader labels checked.
- Color contrast checked.
- Dark mode checked.
- Mobile tap targets checked.

## Operations

- Support inbox ready.
- Moderation owner assigned.
- Incident response owner assigned.
- Analytics dashboard ready.
- Backup/restore plan documented.

## Beta success criteria

- 100 beta users.
- 40% weekly check-in activity.
- 25% community participation.
- Zero unresolved critical safety incidents.
- Clear premium conversion signal.
