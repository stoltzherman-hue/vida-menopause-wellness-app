# Onboarding and User Flows

## Onboarding goals

- Build trust quickly.
- Avoid overwhelming users.
- Capture enough data for personalization.
- Let users skip sensitive answers.
- Separate private health profile from public community profile.

## Onboarding flow

### Step 1: Welcome

Message:

> Vida helps you track symptoms, understand patterns, prepare for appointments, and connect with women who get it.

Actions:

- Continue privately.
- Explore community first.
- Sign in/create account.

### Step 2: Privacy reassurance

Explain:

- Health profile is private by default.
- Community profile is separate.
- Data is not posted automatically.
- AI analysis is optional.

Required consents:

- Terms.
- Privacy policy.
- Health data processing.

Optional consents:

- AI analysis.
- Notifications.
- Marketing.

### Step 3: Stage

Options:

- Perimenopause.
- Menopause.
- Post-menopause.
- Surgical menopause.
- Premature ovarian insufficiency.
- Unsure.

### Step 4: Symptoms

Show categories and allow search.

Default recommended choices:

- Hot flashes.
- Night sweats.
- Sleep disruption.
- Brain fog.
- Mood swings.
- Anxiety.
- Low mood.
- Fatigue.
- Weight/body changes.
- Joint pain.
- Vaginal dryness.
- Low libido.
- Irregular periods.

### Step 5: Health context

Collect optional:

- HRT status.
- Medications.
- Key conditions.
- Hysterectomy/oophorectomy.
- Breast cancer history.
- Osteoporosis/osteopenia.
- Thyroid disease.
- Cardiovascular disease.
- Hypertension.
- Diabetes.
- Migraine.
- Depression/anxiety.

### Step 6: Lifestyle

Collect:

- Sleep quality.
- Exercise frequency.
- Strength training frequency.
- Stress level.
- Alcohol.
- Caffeine.
- Smoking/vaping.
- Diet preference.

### Step 7: Goals

Rank top 3:

- Sleep better.
- Reduce hot flashes/night sweats.
- Improve mood.
- Improve energy.
- Manage weight/body changes.
- Improve brain fog.
- Understand HRT options.
- Prepare for doctor.
- Improve sex/vaginal comfort.
- Build strength/bone health.
- Feel less alone.

### Step 8: Community profile

- Display name.
- Avatar color or icon.
- Visibility level.
- Topics of interest.

### Step 9: First plan

Generate a short plan:

- Top 3 focus areas.
- Tracking setup.
- Suggested first week actions.
- Community suggestions.
- Doctor-prep suggestion if red flags or high severity.

## Key flows

### Daily check-in flow

1. Open app.
2. Tap "Check in".
3. Rate top symptoms.
4. Add sleep/mood/energy.
5. Toggle triggers.
6. Optional journal.
7. Save.
8. Receive one insight or encouragement.

### AI companion flow

1. User opens companion.
2. Selects mode or types freely.
3. App loads minimal context.
4. Safety classifier checks red flags.
5. AI responds.
6. If useful memory emerges, ask permission before saving.

### Community flow

1. User chooses topic.
2. Reads posts.
3. Posts with pseudonym.
4. Gets replies.
5. Can report misinformation or harmful content.

### Doctor report flow

1. Select date range.
2. Choose sections.
3. Generate summary.
4. Edit questions.
5. Export PDF/CSV.
