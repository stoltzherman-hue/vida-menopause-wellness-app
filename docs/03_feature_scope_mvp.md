# MVP Feature Scope

## Free tier

Free must deliver real value and build trust.

Included:

- Anonymous browsing of educational content.
- Account creation with pseudonymous community profile.
- Full community reading, posting, replying, reacting, and reporting.
- Basic Circles access with reasonable limits.
- Daily tracking for up to 5 selected symptoms.
- Basic trends for 30 days.
- Limited AI companion messages per month.
- Basic educational wellness plan.
- Basic reminders.
- Data export in CSV.

## Premium tier

Premium should feel like a private menopause intelligence layer.

Included:

- Unlimited symptom tracking.
- Advanced dashboard and correlation insights.
- AI companion with higher usage limits.
- Voice conversations.
- HRT and medication tracker.
- Advanced reminders and refill alerts.
- Doctor visit reports.
- Personalized wellness plans.
- Full content library.
- Priority access to expert Q&A sessions.
- Advanced Circle matching.

## MVP feature cut decisions

Build now:

- Text AI companion.
- Basic community.
- Daily tracker.
- Medication tracker.
- Reports.

Build next:

- Voice.
- Wearable integrations.
- Expert live events.
- AI-generated content pathways.
- Advanced cohort matching.

Build later:

- Telehealth referrals.
- Clinician portal.
- Employer dashboard without individual health data.
- Native mobile app.
- Lab integrations.
